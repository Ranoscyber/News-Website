Windows Endpoint Protection Manager Vulnerability Leads to Domain Admin Privileges

On August 10, 2025

Cybersecurity researchers have detailed a sophisticated exploit chain targeting Microsoft's Windows Endpoint Protection Manager that enables attackers to escalate privileges from standard user to domain administrator. The vulnerability, tracked as CVE-2025-0234, affects Windows Endpoint Protection Manager versions 10.0 through 11.2. The flaw allows authenticated users to execute arbitrary code with SYSTEM privileges through a race condition in the service's file handling mechanism. Microsoft has released an emergency patch for all affected versions and recommends immediate deployment across enterprise environments.
